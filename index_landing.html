FAQ

Most common questions from photographers.

Trial खत्म होने के बाद क्या होगा?

14 days बाद account auto-disable होगा। Email reminders भेजे जाएंगे ताकि आप plan upgrade करके reactivate कर सको।

Free user में AI Search limit क्यों?

AI search cost बढ़ा सकता है (AWS services). इसलिए trial/free में quota रखा है ताकि bill control रहे।

Super Admin को alerts कैसे मिलेंगे?

Expiry / storage limit / search limit पे system super admin email पर alert भेजेगा + account auto-disable भी कर सकता है।

Google login कैसे होगा?

Cognito में Google IdP जोड़कर Hosted UI / OAuth flow से login होगा। Landing page में button ready है।
